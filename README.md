# courseworksem3
# От градиентного спуска к градиентному потоку

Репозиторий с кодом и экспериментами к курсовой работе.
Исследование поведения градиентного спуска (GD) и стохастического градиентного спуска (SGD) в сравнении с непрерывными моделями (ODE/SDE).

##  Вычислительная среда

Эксперименты проводились на следующей конфигурации:
*   **ОС:** macOS Sequoia (Apple Silicon)
*   **CPU:** Apple M-series (ARM64)
*   **Python:** 3.12 / 3.14
*   **Ключевые библиотеки:** `numpy`, `matplotlib`

## Установка и запуск

Для воспроизводимости результатов используется фиксированное зерно генератора случайных чисел (`SEED = 42`).

### 1. Клонирование репозитория
git clone https://github.com/peskovmatvej06/courseworksem3.git
cd courseworksem3


### 2. Установка зависимостей

pip install -r requirements.txt


### 3. Запуск всех экспериментов (Run All)
Чтобы сгенерировать все графики разом, запустите общий скрипт:

python3 run_all.py

Скрипт создаст в текущей папке файлы: `quadratic.png`, `rosenbrock.png`, `sgd_cloud.png`.

### 4. Запуск по отдельности
Также можно запускать эксперименты независимо друг от друга:

python3 quadratic.py
python3 rosenbrock.py
python3 sgd.py


## Описание экспериментов

1.  **Граница устойчивости (Quadratic):** Проверка теоретического предела $\eta < 2/L$. Файл `quadratic.py`.
2.  **Невыпуклый случай (Rosenbrock):** Сравнение траекторий GD и ODE в сложном овраге. Файл `rosenbrock.py`.
3.  **Стохастика (SGD):** Демонстрация стационарного распределения ("облака") из-за шума в градиентах. Файл `sgd.py`.
